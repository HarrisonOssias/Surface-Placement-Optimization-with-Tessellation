QUICK START - TRAIN YOUR MODEL

OPTION 1: Use the Quick Launcher (EASIEST)

  Then select:
    1 = Quick training (30 min)
    2 = Balanced training (1 hour) [RECOMMENDED]
    3 = Best accuracy (2-3 hours)

OPTION 2: Run Training Directly

  # Install dependencies
  pip install ultralytics pillow pyyaml
  
  # Train model (recommended settings)
  python train_yolo.py --dataset_path "C:\Users\Sarri\Downloads\rgbd-scenes-v2_imgs\rgbd-scenes-v2" --epochs 100 --model_size s
  
  # Your trained model will be saved to:
  # runs\detect\box_detection\weights\best.pt

OPTION 3: Custom Training

  # Nano (fastest, 70-80% accuracy)
  python train_yolo.py --model_size n --epochs 80
  
  # Small (balanced, 80-90% accuracy) [RECOMMENDED]
  python train_yolo.py --model_size s --epochs 100
  
  # Medium (best accuracy, 85-92%)
  python train_yolo.py --model_size m --epochs 150

========================================================================
USING YOUR TRAINED MODEL
========================================================================

After training completes, update box_detector.py to use your model:

METHOD 1: Edit the main() function in box_detector.py

  # Change this line:
  detector = BoxDetector()
  
  # To this:
  detector = BoxDetector(
      use_ml_model=True,
      model_path='runs/detect/box_detection/weights/best.pt'
  )

METHOD 2: Import and use in your code

  from box_detector import BoxDetector
  
  detector = BoxDetector(
      use_ml_model=True,
      model_path='runs/detect/box_detection/weights/best.pt'
  )
  
  boxes = detector.detect(color_frame, depth_frame, kinect_mapper)
